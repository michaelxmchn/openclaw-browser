### 使用方法

1. 启动浏览器服务:
```bash
cd ~/projects/openclaw-browser
npm start
```

2. 使用 Node.js 客户端:
```javascript
const browser = require('./client');
await browser.create();
await browser.navigate('https://example.com');
await browser.screenshot();
await browser.close();
```

3. 使用 REST API:
```bash
curl -X POST http://localhost:3847/api/browser/create \
  -H "Content-Type: application/json" \
  -H "X-API-Key: test" \
  -d '{"headless": true}'
```
