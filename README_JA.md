# OpenClaw Browser 🕸️

> OpenClaw専用のブラウザ。ブラウザ拡張を置き換え、全プラットフォーム自動化を実現

[![MIT License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![GitHub Stars](https://img.shields.io/github/stars/michaelxmchn/openclaw-browser?style=flat)](https://github.com/michaelxmchn/openclaw-browser/stargazers)

## ✨ 特徴

- 🔌 **API制御** - REST APIでOpenClawがブラウザを完全に制御
- 🌐 **全プラットフォーム** - Ubuntu → iOS → Windows
- 🎯 **ヘッドレスモード** - ヘッドレス/ヘッドフル両対応
- 🛡️ **セキュリティ認証** - APIキー認証
- 🔧 **完全な自動化** - ページ操作、フォーム入力、ファイル送受信、スクリーンショット
- ♾️ **全サイト対応** - フルChromiumレンダリングエンジン
