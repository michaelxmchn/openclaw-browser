# OpenClaw Browser - 需求规格文档

## 1. 项目概述

| 项目 | 内容 |
|-----|------|
| **产品名称** | OpenClaw Browser |
| **项目类型** | 浏览器 + 自动化工具 |
| **核心功能** | 替代浏览器扩展，让 OpenClaw 完全控制浏览器实现自动化 |
| **目标平台** | Ubuntu → iOS → Windows |
| **开源协议** | MIT |
