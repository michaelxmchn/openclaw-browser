# OpenClaw Browser 🕸️

> OpenClaw 전용 브라우저. 브라우저 확장 프로그램을 대체하고全 플랫폼 자동화を実現

[![MIT License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![GitHub Stars](https://img.shields.io/github/stars/michaelxmchn/openclaw-browser?style=flat)](https://github.com/michaelxmchn/openclaw-browser/stargazers)

## ✨ 기능

- 🔌 **API 제어** - REST API로 OpenClaw이 브라우저를 완전히 제어
- 🌐 **전체 플랫폼** - Ubuntu → iOS → Windows
- 🎯 **헤드리스 모드** - 헤드리스/헤드폴両 지원
- 🛡️ **보안 인증** - API 키 인증
- 🔧 **완전한 자동화** - 페이지 작업, 양식 입력, 파일 업로드/다운로드, 스크린샷
- ♾️ **모든 사이트 지원** - 완전한 Chromium 렌더링 엔진
